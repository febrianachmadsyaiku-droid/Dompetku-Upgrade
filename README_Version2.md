# 💰 Dompetku v2.0 - Aplikasi Pencatatan Keuangan Pribadi

Dompetku adalah aplikasi pencatatan keuangan pribadi yang dirancang khusus untuk pengguna Indonesia. Versi 2.0 telah ditingkatkan dengan fitur-fitur yang lebih powerful dan komprehensif.

## ✨ Fitur Utama (Versi 2.0)

### 📊 Dashboard Komprehensif
- Total pemasukan, pengeluaran, saldo total, dan target tabung
- Ringkasan saldo per akun/dompet
- Grafik tren 30 hari (pemasukan vs pengeluaran)
- Grafik distribusi pengeluaran per kategori
- Daftar 5 transaksi terbaru

### 🏦 Multi-Account Support (BARU!)
- Membedakan transaksi dari berbagai sumber:
  - 🏦 Bank (BCA, Mandiri, BRI, dll)
  - 📱 E-Wallet (Dana, OVO, GoPay, LinkAja, dll)
  - 💵 Tunai
  - 📈 Investasi
  - 📝 Lainnya
- Tracking saldo per akun otomatis
- Warna badge custom untuk setiap akun
- Ringkasan per akun di dashboard

### 🎯 Target Tabung/Saving Goals (BARU!)
- Buat target untuk membeli sesuatu
- Track progress tabungan secara real-time
- Progress bar visual dengan status warna
- Lihat berapa lagi yang perlu ditabung
- Deadline untuk setiap target
- Edit target kapan saja

### 💳 Manajemen Transaksi
- Catat pemasukan dan pengeluaran dengan detail
- Kategori bawaan yang sudah tersedia
- Tambah kategori custom sesuai kebutuhan
- **Transaksi Berulang**: Pembayaran otomatis harian/mingguan/bulanan/tahunan
- Setiap transaksi dapat dihubungkan ke akun/dompet spesifik
- Edit dan hapus transaksi
- Pencarian dan filter canggih (tipe, kategori, akun, bulan)

### 📈 Manajemen Anggaran
- Tetapkan batas anggaran per kategori per bulan
- Monitor progress anggaran dengan progress bar visual
- Alert otomatis saat mendekati atau melampaui batas
- Color-coded status (hijau, oranye, merah)

### 📋 Laporan Keuangan
- Laporan bulanan lengkap
- Ringkasan total pemasukan, pengeluaran, dan saldo
- Rincian pengeluaran per kategori dengan persentase
- **Rincian transaksi per akun** (NEW!)
- Daftar transaksi detail
- **Export laporan ke format PDF** (NEW!)

### ⚙️ Pengaturan
- Kelola kategori custom (tambah/hapus)
- Backup data ke **file PDF** (NEW!)
- Restore dari file backup JSON
- Hapus semua data

## 🌐 Spesifikasi Teknis

- **Bahasa**: Sepenuhnya dalam Bahasa Indonesia
- **Mata Uang**: Rupiah (Rp) dengan format Indonesia
- **Penyimpanan**: Local Storage (data tersimpan di browser)
- **Privasi**: Data tidak dikirim ke server manapun
- **Offline**: Bisa digunakan tanpa koneksi internet
- **Responsive**: Kompatibel dengan desktop, tablet, dan mobile
- **Export**: PDF untuk laporan dan backup
